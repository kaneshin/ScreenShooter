## Tutorial

```
{
  "data": [{
    "url": "http://example.com/1.0/image",
    "method": "POST",
    "data": {
      "token": "YOUR ACCESS TOKEN",
      "image": "{{_FILE_}}"
    }
  },
  {
    "url": "http://example.org/",
    "method": "PUT",
    "data": {
      "token": "YOUR ACCESS TOKEN",
      "file": "{{_FILE_}}",
      "title": "img.png"
    }
  }]
}
```
